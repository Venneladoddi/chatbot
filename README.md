# chatbot  # 💰 Expense Tracker using Python

A simple command-line based Expense Tracker application built using Python.

This project helps users record, view, and search their daily expenses.

## 🚀 Features

- Add new expenses
- Store expense details permanently
- View complete expense history
- Calculate total spending
- Search expenses by category
- Uses JSON file for data storage


## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- datetime module
